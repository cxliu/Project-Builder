M-Build Future Enhancements


This is a list of things that would be improvements in the direction that M-Build would like to go. None of these are promised, but rather these are listed to give an idea of the types of things that are perceived as things that would be consistent with the purpose of M-Build.

  • Backends:
    • A new template language for Makefiles
    • An XCode project generator
    • Ninja project generator?
    • Eclipse project generator?
  • Deprecations
    • @priority
  • Documentation
    • Improvements to output of create_doc
    • Complete reference guide
    • Debugging
    • More helpful error messages
    • Improvements to query_manifest
  • API
    • The M-Build python API could do with a lot of improvement
  • Projects
    • Support for projects not managed by M-Build
